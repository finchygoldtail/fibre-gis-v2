Simple QGIS use:
1. Open Project > New Print Layout.
2. Set page size to A4 landscape.
3. Add Picture and choose alistra-a4-job-pack-layout.svg.
4. Add Map or Picture screenshot over the main route frame.
   Recommended frame: X 12mm, Y 32mm, W 210mm, H 162mm.
   Use a transparent map background if you want the dashed frame to remain visible.
5. Keep the selected 96F route blue, with 12F/48F routes as context where needed.
6. Export the layout as PDF.

Simple screenshot workflow:
1. In QGIS, zoom to the route and export/screenshot the map.
2. In the A4 layout, add the template SVG as the bottom Picture item.
3. Add your route screenshot as a Picture item over the route frame.
4. Keep the side key/template visible and export to PDF.

Legacy manual map workflow:
1. Open Project > New Print Layout.
2. Set page size to A4 landscape.
3. Add Picture and choose alistra-a4-job-pack-layout.svg.
4. Add Map and draw it over the main map frame area.
5. Use the legend SVG if you want a fixed contractor-style legend instead of a live QGIS legend.

Recommended:
Use alistra-a4-job-pack-layout.svg as a visual guide/title block, then place the live QGIS map frame over the MAP FRAME area.
